# MyAnimeRecs

#### This is a simple anime recommendation website, created using [ReactJS](https://react.dev/) as the Frontend Library and [Flask](https://flask.palletsprojects.com/en/2.1.x/) as the Backend Framework.
#### This project is made for Proof of Concept for my group thesis: ***Building an Anime Recommendation System using Machine Learning Methods with Sentiment Analysis***
#### In this website, user can browse animes, add them to the watchlist, and receive recommendations based on their input. For details about the algorithm, refer to [this section](/server/recommendation.py).
#### I also used [MongoDB](https://www.mongodb.com/) to store data, [Tailwind CSS](https://tailwindcss.com/) for styling, [React Router](https://reactrouter.com/en/main) for client-side routing and [React Query](https://tanstack.com/query/v3/) to increase data fetching performance.
